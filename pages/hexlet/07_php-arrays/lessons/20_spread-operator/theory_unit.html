Spread-оператор

Spread-оператор позволяет распаковывать элементы массива внутри другого. С его помощью обычно копируют или сливают массивы.

Представьте, что нам нужно определить массив, добавив туда элементы из другого массива. Такая задача часто встречается при работе со значениями по умолчанию:

<?php

$russianCities = ['moscow', 'kazan'];
$cities = ['milan', 'rome', ...$russianCities];
// ['milan', 'rome', 'moscow', 'kazan']

// Массив $russianCities при этом никак не меняется

// То же самое без spread-оператора
$cities = array_merge(['milan', 'rome'], $russianCities);

... в данном случае spread-оператор. Он растянул массив, добавив все его элементы в новый массив. Spread-оператор можно использовать в любой части массива. Например, мы можем дополнить исходный массив не справа, а слева:

<?php

$cities = [...$russianCities, 'milan', 'rome'];
// ['moscow', 'kazan', 'milan', 'rome']

// То же самое без spread-оператора
$cities = array_merge($russianCities, ['milan', 'rome']);

И даже посередине:

<?php

$cities = ['milan', ...$russianCities, 'rome'];
// ['milan', 'moscow', 'kazan', 'rome']

// Без spread-оператора подобный код нельзя выразить одной операцией

Spread-оператор работает с любым количеством массивов:

<?php

$russianCities = ['moscow', 'kazan'];
$ukrainianCities = ['kiev', 'odessa'];
// слияние двух массивов
$cities = [...$russianCities, ...$ukrainianCities];
// ['moscow', 'kazan', 'kiev', 'odessa']

// То же самое без spread-оператора
$cities = array_merge($russianCities, $ukrainianCities);
