Операции

Вы знаете и умеете применять основные операции, связанные с примитивными типами данных.

<?php

// Арифметические операции
1 + 5 - 8 * 6 / 2 ** 3 % 4;

// Логические операции
true || false && !true

// Создание переменных
$str = 'i am a string';

// Обращение к символам строки
$str[0]; // i
print_r("{$str}"); // => i am a string
print_r($str); // => i am a string

Условные конструкции

<?php

if ($a > $b) {
    echo 'a больше, чем b';
} elseif ($a === $b) {
    echo 'a равен b';
} else {
    echo 'a меньше, чем b';
}

$result = $a > $b ? 'yes' : 'no';

switch ($i) {
    case 0:
        echo "i равно 0";
        break;
    case 1:
        echo "i равно 1";
        break;
    case 2:
        echo "i равно 2";
        break;
    default:
       echo "i не равно 0, 1 или 2";
}

Циклы

<?php

$i = 1;
while ($i <= 10) {
    echo $i++;
}

for ($i = 1; $i <= 10; $i++) {
    echo $i;
}
