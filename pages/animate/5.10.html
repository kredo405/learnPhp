Базовая теория

Глава 5.10. Функции в WebGL

В этой главе приведём основные встроенные в GLSL функции, которыми можно пользоваться при написании шейдеров.

Угловые и тригонометрические функции

Перевод значений между радианами и градусами

float aDeg = ...		// значение угла в градусах 

float aRad = radians(aDeg);	// -> в радианах
float a = degrees(aRad);	// -> обратно в градусы

Синус и косинус

Для программного рисования и анимации нельзя придумать лучших друзей, чем синус и косинус. В этом уже можно было убедиться при изучении глав о покадровой анимации. В WebGL они тоже используются.

float a = ...		// значение угла в радианах

float x = cos(a);
float y = sin(a);

Параметрическое описание круга с помощью синуса и косинуса.

Входящие параметры

degrees, radianssin, cos и другие тригонометрические функции принимают на вход не одно только дробное число, но и векторы со значениями в компонентах. В этом случае на выходе они также имеют векторы:

vec4 vecA = vec4(...)	// вектор со значениями углов в радианах

vec4 vecX = cos(v);	// вектор с рассчитанными для каждого компонента косинусами
vec4 vecY = sin(v);	// вектор с рассчитанными для каждого компонента синусами

Другие тригонометрические и математические функции

Синтаксис Описание Диапазон выходных значений
tan (a) тангенс угла любое значение
asin (a) угол, значение синуса которого равно входящему параметру a [-PI/2, PI/2], не определён, если abs (a) > 1
acos (a) угол, значение косинуса которого равно входящему параметру a [0, PI], не определён, если abs (a) > 1
atan (y_devided_by_x) угол, значение тангенса которого равно входящему параметру a [-PI/2, PI/2]
atan (y, x) угол, значение тангенса которого равно отношению первого параметра ко второму [-PI, PI], не определён, если x=0 и y=0. Знаки x и y определяют, в каком квадранте находится угол

Экспоненциальные функции

Синтаксис Описание Диапазон выходных значений
exp (x) экспонента x [0, ∞)
sinh (x) гиперболический синус x:
(exp(x) - exp(-x))/2
cosh (x) гиперболический косинус x:
(exp(x) + exp(-x))/2
tanh (x) гиперболический тангенс x:
(exp(x) - exp(-x))/(exp(x) + exp(-x))

Справка: Гиперболические функции

Другие удобные функции для работы с числами

Синтаксис Описание Диапазон выходных значений
abs (x) абсолютное значение x (float или int): x, если x >= 0, −x, если x < 0 [0, ∞)
sign (x) знак величины x (float или int): 1.0, если x > 0, 0.0, если x = 0, −1.0, если x < 0 [−1.0, 1.0]
round (x) ближайшее к x целых чисел (0.5 округлится до 1.0, а −0.5 до −1.0) любое целочисленное значение
roundEven (x) ближайшее к x целых чисел (дробная часть 0.5 округлится до ближайшего чётного целого числа) любое целочисленное значение
floor (x) меньшее из ближайших к x целых чисел любое целочисленное значение
ceil (x) большее из ближайших к x целых чисел любое целочисленное значение
trunc (x) ближайшее целое число, чьё абсолютное значение не больше, чем абсолютное значение x любое целочисленное значение
fract (x) x - floor(x) [−1.0, 1.0]
mod (x, float y) модуль: x - y * floor(x/y), где x — float, vec{2,3,4}, а float y, применяется к каждому компоненту x любое значение
mod (x, vec{2,3,4} y) модуль: x - y * floor(x/y), где x — vec{2,3,4}, а y — vec{2,3,4} той же размерности, компоненты y применяются к компонентам x соответственно любое значение
min (x, float y) минимальное из значений, где x — float, vec{2,3,4}, а float y, сравнивается с каждым компонентом x любое значение
min (x, vec{2,3,4} y) минимальное из значений, где x — vec{2,3,4}, а y — vec{2,3,4} той же размерности, компоненты y сравнивается с компонентами x соответственно любое значение
max (x, float y) максимальное из значений, где x — float, vec{2,3,4}, а float y, сравнивается с каждым компонентом x любое значение
max (x, vec{2,3,4} y) максимальное из значений, где x — vec{2,3,4}, а y — vec{2,3,4} той же размерности, компоненты y сравнивается с компонентами x соответственно любое значение
clamp (x, minVal, maxVal) min(max(x, minVal), maxVal) значение в [minVal, maxVal]

Shaping-функции

Используются для сглаживания границ между цветами при рисовании во фрагментном шейдере.

Синтаксис Описание
mix (x0, x1, a) линейная интерполяция на отрезке [x0, x1]. a — [0.0, 1.0], относительное положение результирующего значения на отрезке
step (edge, x) ступенчатая интерполяция: возвращает 0.0, если x < edge, иначе 1.0
smoothstep (x) эрмитова интерполяция на отрезке [x0, x1]

Линейная интерполяция даёт плавный переход между цветами.

Ступенчатая интерполяция даёт резкую границу.

Эрмитова интерполяция даёт плавный переход, однако не такой размытый, как линейная интерполяция.

Подробнее о shaping-функциях: https://www.iquilezles.org/www/articles/functions/functions.htm, https://thebookofshaders.com/05/
